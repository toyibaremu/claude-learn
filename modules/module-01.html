⚙️ Installation & Setup

Get Claude Code running on your computer in under 15 minutes — step by step, no assumptions made.

😌
Don't panic about the terminal The terminal (also called the command line or shell) sounds intimidating, but we'll walk through every step. If you've ever typed a command in Stata or R's console, you already understand the core concept. It's just another place to type instructions.

What you'll need

  • A computer running Mac, Windows, or Linux
  • An internet connection
  • An Anthropic account (free to create at console.anthropic.com)
  • ~$5–10 to add to your API account (or a Claude Pro subscription for the web option)
  • About 15–30 minutes

Option A: Claude Code on the web (easiest)

If you have a Claude.ai Pro, Team, or Max subscription, you can use Claude Code directly in your browser — no installation required.

⚠️
Web vs. Terminal: key difference The web version is great for getting started, but the terminal version is more powerful — it can directly read and write files on your computer without you uploading them, run code, and integrate with your whole project folder automatically. Most power users prefer the terminal. This course covers both. Start with whichever feels comfortable.

Option B: Claude Code in your terminal (recommended for power use)

The terminal version of Claude Code requires installing two things:

  1. Node.js — a runtime environment that Claude Code needs to run
  2. Claude Code itself — installed using npm (a package manager that comes with Node.js)
📖
Jargon check: What is npm? npm stands for "Node Package Manager". Think of it like Stata's ssc install or R's install.packages() — it's a way to download and install software tools from the internet with a single command. You won't need to understand it deeply; just use it to install Claude Code.

Installing on Mac

Step 1: Open your Terminal

Press Cmd + Space, type "Terminal", and press Enter. You'll see a window with a blinking cursor.

Step 2: Install Node.js

Go to nodejs.org, download the LTS version, and run the installer. Or, if you have Homebrew:

# If you have Homebrew (a Mac package manager)
brew install node

# Check it worked — you should see a version number like "v22.x.x"
node --version

Step 3: Install Claude Code

npm install -g @anthropic-ai/claude-code

The -g flag means "install globally" — so you can use it anywhere on your computer, not just in one folder.

Step 4: Start Claude Code

# Navigate to your project folder first
cd ~/Documents/my-research-project

# Then start Claude Code
claude

Installing on Windows

💡
Recommended: Use WSL (Windows Subsystem for Linux) Claude Code works best on Windows through WSL — a built-in Windows feature that gives you a Linux environment. It's easier than it sounds. Follow these steps:

Step 1: Enable WSL

Open PowerShell as Administrator (search "PowerShell", right-click → "Run as administrator") and run:

wsl --install

Restart your computer when prompted.

Step 2: Open Ubuntu (WSL)

Search for "Ubuntu" in your Start menu. The first time you open it, set up a username and password.

Step 3: Install Node.js inside WSL

# Install Node.js version manager (nvm)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Reload your shell settings
source ~/.bashrc

# Install the latest LTS version of Node.js
nvm install --lts

# Check it worked
node --version

Step 4: Install Claude Code

npm install -g @anthropic-ai/claude-code
💡
Alternative: Native Windows (without WSL) Claude Code does work on Windows natively. Download Node.js from nodejs.org, install it, then open Command Prompt or PowerShell and run npm install -g @anthropic-ai/claude-code. WSL is just recommended for a smoother experience.

Installing on Linux

# Install Node.js (Ubuntu/Debian)
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# Install Claude Code
npm install -g @anthropic-ai/claude-code

# Verify
claude --version

Getting your API key

An API key is like a password that lets Claude Code connect to Anthropic's servers on your behalf. You need one to use the terminal version of Claude Code.

Create API key
Under "API Keys" section
📋

When you first run claude, it will ask for your API key. Paste it in and press Enter. Claude Code stores it securely — you only need to do this once.

🔒
Keep your API key secret! Your API key is like a password with a credit card attached. Never share it, never paste it into a document or email, never commit it to GitHub. If you accidentally expose it, go to console.anthropic.com immediately and delete it, then create a new one.

Understanding cost

Claude Code charges per use, measured in tokens (roughly ¾ of a word). You pay for what you use, like a prepaid phone plan.

Usage patternEstimated monthly cost
Light use (a few sessions/week)$3–10
Regular use (daily, moderate sessions)$10–40
Heavy use (all-day coding assistant)$40–100+
Claude Code Max subscription$100/month flat
💰
Tip for staying within budget Set a spending limit at console.anthropic.com → "Billing" → "Usage limits". Start with $10 and increase as you learn. Most researchers find $15–30/month covers daily use comfortably.

Verify your installation

Run these commands to confirm everything is working:

# Check Claude Code is installed
claude --version

# Start a session (creates a test folder first)
mkdir ~/claude-test && cd ~/claude-test
claude

# Claude Code will start and show a prompt like:
>

If you see the > prompt, you're in! Type "hello" to test it. Claude should respond — and that's Claude Code running on your computer.

To exit Claude Code, type /exit or press Ctrl+C.

🛠️
Troubleshooting: common issues
  • "command not found: claude" — Node.js might not be installed correctly. Try closing and reopening the terminal.
  • "permission denied" — On Mac/Linux, try: sudo npm install -g @anthropic-ai/claude-code
  • API key not working — Make sure you added billing credits at console.anthropic.com
  • Still stuck? See the official docs or post in the GitHub issues.
🎯

Module 1 Exercise: Get it running!

Complete all three steps:

  1. Install Claude Code using the instructions for your operating system above.
  2. Get your API key from console.anthropic.com and connect it.
  3. Navigate to a folder on your computer (it can be any folder — your Desktop, or a project folder), start Claude Code, and ask it: "What files are in this folder and what do you think they're for?"

You've succeeded when Claude Code describes your files back to you. Congratulations — you have a working Claude Code setup!
